# UniTo - LFT - Progetto di laboratorio - 2019/2020

## Run an exercise
```shell
$ cd sect{01..05}
$ cd ex{01..}
$ make [SRC="path/to/source.txt"]
```
If no `SRC` argument is passed, the source text file that within the exercise directory will be used.


## Clean a section
```shell
$ cd sect{01..05}
$ cd ex{01..}
$ make clean
```
